#Introdução

Microsserviço responsável por lidar com autenticação e autorização.
O serviço foi totalmente desenvolvido seguindo a especificação netstandard 2.0, com a maior parte dos binários compilados para netcore2.0.


#Compilar e Testar

Para compilar e executar o projeto, é necessário ter o .NET Core SDK 2.0 instalado.
Você pode fazer o download em http://dot.net

Para executar, basta usar o comando:

dotnet run
